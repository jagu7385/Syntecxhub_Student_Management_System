# Student Management System (CLI)

A simple Command-Line Interface (CLI) based Student Management System built using Python.
This project allows users to manage student records including Name, ID, and Grade with file persistence using JSON.

It demonstrates Object-Oriented Programming (OOP) concepts and file handling in Python.

 # Features

 Add new student

 Update existing student

 Delete student record

 View all students

 Data stored in JSON file (persistent storage)

 Unique Student ID validation

 Clean formatted console output

 # Technologies Used

Python 3

JSON (File Storage)

OOP Concepts (Classes, Constructors, Methods)

File I/O Operations
